## Overview
HealthMate is a Python-based health and fitness application that helps users collect, calculate, and categorize health-related data. It provides valuable information such as BMI, BMR, body fat percentage, and ideal weight range. Additionally, HealthMate offers exercise and diet recommendations based on user categories and goals.

## Features
- Collect user data, including name, age, gender, weight, and height.
- Calculate essential health metrics, including BMI, BMR, body fat percentage, and ideal weight range.
- Categorize users based on calculated metrics.
- Set health and fitness goals.
- Provide exercise and diet recommendations tailored to user categories and goals.
- Generate a detailed health report in a Word document.

## Usage
1. Run `main.py` to start the HealthMate application.
2. Follow the prompts to enter your personal information, health goals, and other details.
3. HealthMate will calculate your health metrics, categorize you, and provide recommendations.
4. A health report will be generated and saved as a Word document.

## Files Included
- `main.py`: The main Python script for the HealthMate application.
- `user_data.csv`: A CSV file that stores user data.
- `appdata.xlsx`: An Excel file containing exercise and diet data.
- `README.md`: This README file.

## Prerequisites
- Python 3.11
- The following Python packages: `csv`, `docx`, `pandas`
